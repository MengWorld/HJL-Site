## 技术栈

- 前端界面框架: Bootstrap
- Web 前端框架: Vue3.js
- 后端: Express.js
- 数据库: SQLite3

## 其他

- Vue Router
- SCSS
- [paletton](https://paletton.com/)

## 关于

1. 贺峻霖是一名青年主持人。
2. 本项目旨在提供一个灵活的平台，用于汇总和展示任何值得关注的个体的经历和成就。
3. 内容更新和管理通过 CMS 进行，支持社区成员的贡献和维护。
